# 어썸 클로드 스킬 탐색기 (Awesome Claude Skills Explorer)

이 문서는 사용자가 추가한 `ComposioHQ/awesome-claude-skills` 저장소의 주요 스킬들을 한국어로 설명합니다. 에이전트 스킬 매니저 UI가 영어로 되어 있어 찾기 힘들 때, 이 문서를 참조하여 원하는 스킬의 **영어 이름**을 확인하고 설치하세요.

---

## 📂 문서 처리 (Document Processing)
파일 생성, 편집, 데이터 추출 및 변환에 유용한 스킬들입니다.

| 한글 이름 | 영어 이름 (UI 검색용) | 상세 설명 |
| :--- | :--- | :--- |
| **Word 문서 마스터** | `docx` | Word 문서 생성, 편집, 분석. 변경 내용 추적 및 서식 지정 지원. |
| **PDF 도우미** | `pdf` | PDF 텍스트/표/메타데이터 추출, PDF 병합 및 주석 추가. |
| **PPT 제작기** | `pptx` | 슬라이드 생성, 레이아웃 조정, 템플릿 기반 발표 자료 제작. |
| **엑셀 전문가** | `xlsx` | 수식, 차트, 데이터 변환 등 복잡한 스프레드시트 조작. |
| **이북 변환기** | `Markdown to EPUB` | 마크다운 문서를 전문적인 EPUB 이북 파일로 변환. |

## 💻 개발 및 코드 도구 (Development & Code)
코딩 자동화, 인프라 관리 및 테스트에 특화된 스킬들입니다.

| 한글 이름 | 영어 이름 (UI 검색용) | 상세 설명 |
| :--- | :--- | :--- |
| **UI 아티팩트 빌더** | `artifacts-builder` | React, Tailwind, shadcn/ui를 사용한 고성능 UI 컴포넌트 제작. |
| **브라우저 자동화** | `Playwright Automation` | Playwright를 이용한 웹 앱 테스트 및 브라우저 매크로 수행. |
| **변경 로그 생성기** | `Changelog Generator` | Git 커밋 내역을 바탕으로 사용자용 업데이트 로그 자동 생성. |
| **프롬프트 엔지니어** | `prompt-engineering` | Anthropic의 권장 프롬프트 작성 기법 및 최적화 방법 안내. |
| **스킬 크리에이터** | `Skill Creator` | 새로운 클로디 스킬을 효과적으로 만드는 워크플로우 가이드. |

## ✍️ 통계 및 글쓰기 (Communication & Writing)
콘텐츠 제작, 리서치 및 분석을 돕는 스킬들입니다.

| 한글 이름 | 영어 이름 (UI 검색용) | 상세 설명 |
| :--- | :--- | :--- |
| **웹 기사 추출기** | `article-extractor` | 웹 페이지에서 본문 텍스트와 메타데이터만 깔끔하게 추출. |
| **브레인스토밍** | `brainstorming` | 아이디어를 구체적인 디자인이나 계획으로 발전시키는 문답형 가이드. |
| **회의 분석기** | `Meeting Insights` | 회의 녹취록을 분석하여 발언 비중, 핵심 패턴 등 도출. |
| **맞춤형 이력서 생성** | `Tailored Resume` | 채용 공고에 맞춰 최적화된 이력서 및 자기소개서 생성. |

## 💡 ERP 등록 매크로 프로젝트 추천 스킬
사용자님의 현재 프로젝트(`ERP등록메크로`)에 특히 도움이 될 만한 스킬들입니다:

1.  **`xlsx` (엑셀 전문가)**: ERP에 등록할 데이터를 엑셀에서 정리하거나 변환할 때 필수적입니다.
2.  **`Playwright Automation` (브라우저 자동화)**: 웹 기반 ERP라면 직접 화면을 조작하여 데이터를 입력하는 매크로를 짤 때 매우 유용합니다.
3.  **`docx` (Word 문서 마스터)**: 작업 결과 보고서나 매뉴얼을 자동으로 워드 파일로 만들 때 좋습니다.
4.  **`Skill Creator` (스킬 크리에이터)**: 나만의 전용 'ERP 등록 스킬'을 직접 만들고 싶을 때 가이드를 제공합니다.

---

### 사용하는 방법
1. 에이전트 스킬 매니저 패널을 엽니다.
2. `ComposioHQ/awesome-claude-skills` 내에서 위 표의 **영어 이름**을 찾습니다.
3. 해당 항목 옆의 체크박스를 클릭하고 **Install**을 누르세요.

> [!NOTE]
> 찾으시는 다른 기능이 있다면 저에게 "~~한 기능을 하는 스킬이 저 레포에 있어?"라고 물어봐 주세요. 제가 즉시 찾아보고 한국어로 설명해 드릴게요!
