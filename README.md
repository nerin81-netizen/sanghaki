# � QuickMoney: Smart Financial Manager

웹 기술(HTML, CSS, JS)만을 사용하여 제작된 심플하고 직관적인 **개인 가계부 애플리케이션**입니다.

## ✨ 주요 기능
- **직관적인 내역 기록**: 수입과 지출을 한눈에 파악하고 빠르게 입력
- **카테고리 커스텀**: 나만의 소비 습관에 맞춘 카테고리 관리
- **월별 시각화**: 달력 및 분석 그래프를 통한 소비 패턴 분석
- **완벽한 프라이버시**: 모든 데이터는 서버가 아닌 **로컬 스토리지**에만 저장되어 안전합니다.
- **반복 지출 자동화**: 매달 나가는 고정 지출을 자동으로 등록하는 스마트 기능

## � 기술 스택
- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Storage**: Browser LocalStorage
- **Design**: Responsive Web Design (Mobile First)
