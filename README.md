# QuickMoney - 개인 가계부 프로젝트

웹 기술(HTML, CSS, JS)을 사용하여 제작된 심플하고 직관적인 개인 가계부 애플리케이션입니다.

## 주요 기능
- 수입 및 지출 내역 기록
- 카테고리별 내역 관리
- 월별 통계 및 시각화
- 로컬 스토리지를 이용한 데이터 보존
- 반응형 웹 디자인 지원

## 실행 방법
1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/사용자명/QuickMoney.git
   ```
2. `index.html` 파일을 브라우저에서 엽니다.

## 기술 스택
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser LocalStorage
- **Design**: Modern, Responsive Design
